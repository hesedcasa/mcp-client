# Usage

<!-- usage -->
```sh-session
$ npm install -g @hesed/mcp-client
$ mcp-client COMMAND

$ mcp-client (--version)
@hesed/mcp-client/0.1.0 darwin-arm64 node-v22.14.0
$ mcp-client --help [COMMAND]
USAGE
  $ mcp-client COMMAND
...
```
